
[[jkube:undeploy]]
=== *{task-prefix}Undeploy*

This task is for deleting the kubernetes resources that you deployed via the  <<jkube:apply, {task-prefix}Apply>> or <<jkube:deploy, {task-prefix}Deploy>> tasks.

It iterates through all the resources generated by the <<jkube:resource, {task-prefix}Resource>> task and deletes them from your current {cluster} cluster.

[source, sh, subs="+attributes"]
----
gradle {task-prefix}Undeploy
----
